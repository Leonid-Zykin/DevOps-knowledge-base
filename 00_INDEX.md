## How to Use

- **Поиск по тегам**: во всех файлах есть общий тег `devops` и тег по разделу.
- **Навигация по ссылкам**: переходите по Obsidian‑ссылкам `[[...]]` между разделами.
- **Практический фокус**: все заметки в стиле шпаргалок с командами и примерами.

> ⚡ Tip: Используйте быстрый поиск по Obsidian (Ctrl+O / Ctrl+P), набирая как команды (`kubectl get`), так и названия файлов (`terraform-state`), чтобы моментально прыгать в нужную шпаргалку.
